## 網頁功能

### 1. 今年以來績效排名 (YTD)
- 位於 Header 下方、分頁按鈕上方
- 同時讀取所有 6 檔 ETF 的 data JSON，依 YTD 由高到低排序
- 顯示名次（🥇🥈🥉）、ETF 代號、名稱、YTD %
- 點擊任一 ETF pill 可直接切換至該 ETF 的持股清單

### 2. 持股清單 (Holdings Tab)
- ETF 選擇器（下拉選單）切換標的
- 顯示：序號、股票代號/名稱、股數、股價、持股比例（含前後比較）、狀態（新增/出清/加碼/減碼）、加減碼股數、加減碼金額
- 加減碼金額欄可點擊排序（降冪 → 升冪 → 預設）
- 顯示資料日期（最新交易日）及最後更新時間
- Header 顯示：經理人、ETF 股價、YTD 績效

### 3. ETF 持股交叉比對 (Cross Tab)
- 彙整所有 6 檔 ETF 持股，找出被 **2 檔以上** ETF 同時持有的標的
- 顯示：序號、股票代號/名稱、持有的 ETF 列表（含各 ETF 中的持股比例）、持有 ETF 數量
- 持有 ETF 數欄可點擊排序

## 資料來源與更新邏輯

### 類型 A：ezmoney.com.tw（00981A）
- Playwright 開啟網頁，點擊下載 XLSX
- 讀取 XLSX，驗證民國日期欄位是否為前一交易日
- 從第 19 行起解析持股資料

### 類型 B：Nomura JSON API（00980A、00985A）
- `POST https://www.nomurafunds.com.tw/API/ETFAPI/api/Fund/GetFundAssets`
- Payload：`{"FundID": "00985A", "SearchDate": "YYYY-MM-DD"}`
- 直接解析 JSON，無需 Playwright

### 類型 C：fhtrust.com.tw 直接下載（00991A）
- `GET https://www.fhtrust.com.tw/api/assetsExcel/ETF23/{YYYYMMDD}`
- 回傳 XLSX，直接解析，無需 Playwright

### 類型 D：capitalfund.com.tw（00992A、00982A）
- Playwright 開啟 Angular 頁面
- 必須用 JS `HTMLInputElement.prototype.value` setter + `dispatchEvent` 設定日期（Angular 雙向綁定，不能用 `.type()`）
- **日期偏移**：在日期選擇器填入 run_date（當天），實際取得的是前一交易日的資料
- 產品 ID：00992A = 500，00982A = 399

### 通用流程（所有腳本）
1. 計算前一交易日（跳過週末）
2. 若 `holdings/` 已有該日快照則跳過
3. 抓取今日持股，儲存快照 JSON
4. 讀取最近一筆歷史快照作為比較基準
5. 用 yfinance 取得每檔股票當日收盤價（.TW 或 .TWO）
6. 計算差異（股數、金額）、YTD、ETF 股價
7. 產生 `data_{CODE}.json`
8. `git add -A && git commit && git push`（觸發 GitHub Pages 更新）
9. 發送 Telegram 通知

## Telegram 通知設定

憑證儲存於 `ETF_Tracker/.env`（不進 git）：

```
TELEGRAM_BOT_TOKEN=你的BotToken
TELEGRAM_CHAT_ID=你的ChatID
```

通知內容包含：資料日期、ETF 股價、YTD、持股數量、加碼/減碼/新增/出清各檔數量及明細（以張數顯示，1張＝1000股）。

## 新增 ETF 步驟

1. **確認資料來源類型**（A/B/C/D，見上方說明）
2. **複製最相近的腳本**重新命名為 `check_and_update_{新代號}.py`
3. 修改腳本頂部：`ETF_CODE`、`MANAGER`、API URL / Product ID
4. **index.html**：在 `<select id="etf-selector">` 新增 `<option>`
5. **app.js**：在 `ALL_ETFS` 和 `ETF_LIST` 兩個陣列中新增 `{ id, name }`
6. 執行腳本產生初始 `data_{CODE}.json`
7. Git push，GitHub Pages 約 1-5 分鐘後生效
8. 工作排程器新增排程

## 常見問題

| 問題 | 原因 | 解法 |
|------|------|------|
| Angular 日期選擇器填值無效 | `.type()` 不觸發雙向綁定 | 用 JS prototype setter + dispatchEvent |
| capitalfund 日期偏移一天 | 選 T 日返回 T-1 資料 | 腳本已處理，勿調整 |
| Telegram 400 Bad Request | 股票名稱含 HTML 特殊字元 | 勿使用 parse_mode=HTML |
| yfinance 無法取得股價 | 未上市或休市 | 依序嘗試 .TW、.TWO，失敗回傳 0.0 |
| GitHub Pages 未更新 | 未 push 或傳播延遲 | 等待 1-5 分鐘，確認 git push 成功 |
